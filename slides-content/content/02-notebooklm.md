## Creating a Notebook

**Step-by-step:**

1. Go to [notebooklm.google.com](https://notebooklm.google.com)
2. Click "New Notebook"
3. Add sources (upload, paste URLs, or do a Fast/Deep Research)
4. Wait for processing
5. Start asking questions!

**Pro tip:** Use clear document titles for better retrieval
